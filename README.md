## Getting Started

First, install package:

```bash
pnpm install
# or
npm install
# or
yarn install
```

Now run the development server :

```bash
pnpm run dev
# or
npm run dev
# or
yarn dev
```

Open [http://localhost:5173](http://localhost:5173)
